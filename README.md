# Data Professionals Survey Dashboard (Power BI)

## Overview

This project presents an interactive Power BI dashboard built using survey data from data professionals across different countries and job roles.

The dashboard provides insights into salary trends, job satisfaction, favorite programming languages, and demographic information through interactive visualizations.

## Tools Used

- Power BI
- Power Query
- Microsoft Excel

## Dashboard Features

- Average Salary by Job Title
- Average Salary by Gender
- Country-wise Distribution
- Favorite Programming Languages
- Work-Life Balance Rating
- Salary Satisfaction Analysis
- Interactive Filters and Slicers

## Skills Demonstrated

- Data Cleaning
- Data Transformation
- Data Modeling
- Data Visualization
- Dashboard Design
- Interactive Reporting

## Dataset

Survey responses from data professionals.

## Files

- Data Professionals Survey Dashboard.pbix
- Power BI Final Project.xlsx

## Key Insights

The dashboard helps identify salary trends, programming language preferences, and satisfaction levels among data professionals across different regions and job roles.
